## 々

叠字符号 々{NoMa} 表示重复前一个字 <br />
在输入法中可以输入 `noma` 来打出 <br />
它的读音是前一个字符的读音 <br />

e.g. <br />
```text
凛凛{RinRin} == 凛々{RinRin}
時時         == 時々
赤裸裸       == 赤裸々
複複複線     == 複々々線

```
