# 句-01




---
## は{Ha / Wa}

`は` 做助词时读作 `は{Wa}`


e.g. <br />
```text
私は誰[わたしわだれ] == 我是谁

```




---
## の{No}

`の` 用于连接名词与名词 <br />
与汉语里的 `的` 有一些区别 <br />


---
## 的{TeKi}

使形容词变为形容动词(な形容词)


e.g. <br />
```
科学的方法
平和的解決
政治的発言
```
